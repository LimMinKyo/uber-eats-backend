# Uber Eats Clone(Backend)

## Introduction

- 기술스택: Nest.js, Typescript, TypeORM, GraphQL, Jest, PostgreSQL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
